# 04. 배열과 포인터

## 학습 목표

- 배열의 개념과 사용법 이해
- 포인터의 개념과 사용법 이해
- 동적 메모리 할당 학습
- 배열과 포인터의 관계 이해

## 학습 내용

### 1. 배열

- 1차원 배열
- 2차원 배열
- 배열의 초기화
- 배열과 함수

### 2. 포인터 기초

- 포인터의 개념
- 주소 연산자 (&)
- 역참조 연산자 (\*)
- 포인터 연산

### 3. 동적 메모리

- new와 delete
- 동적 배열
- 메모리 누수 방지

### 4. 문자열과 포인터

- C 스타일 문자열
- 문자 배열
- string 클래스

## 예제 파일

- `01_arrays.cpp` - 배열 기초
- `02_multidimensional_arrays.cpp` - 다차원 배열
- `03_pointers.cpp` - 포인터 기초
- `04_dynamic_memory.cpp` - 동적 메모리
- `05_arrays_and_pointers.cpp` - 배열과 포인터

## 연습 문제

1. 배열에서 최댓값과 최솟값을 찾는 프로그램
2. 2차원 배열로 행렬의 곱셈 구현
3. 동적 배열로 학생 점수 관리 프로그램
4. 포인터를 이용한 문자열 처리
