# 03. 함수

## 학습 목표

- 함수의 개념과 필요성 이해
- 함수 정의와 호출 방법 학습
- 매개변수와 반환값 이해
- 함수 오버로딩과 재귀 함수 학습

## 학습 내용

### 1. 함수 기초

- 함수의 정의와 호출
- 매개변수(parameter)와 인자(argument)
- 반환값(return value)
- void 함수

### 2. 함수 고급

- 기본 매개변수
- 함수 오버로딩
- 인라인 함수
- 재귀 함수

### 3. 참조와 포인터

- 값에 의한 전달 (pass by value)
- 참조에 의한 전달 (pass by reference)
- 포인터를 이용한 전달

## 예제 파일

- `01_basic_functions.cpp` - 기본 함수
- `02_parameters.cpp` - 매개변수와 반환값
- `03_function_overloading.cpp` - 함수 오버로딩
- `04_recursion.cpp` - 재귀 함수
- `05_pass_by_reference.cpp` - 참조에 의한 전달

## 연습 문제

1. 두 수의 최대공약수를 구하는 함수 작성
2. 팩토리얼을 구하는 재귀 함수 작성
3. 배열의 평균을 구하는 함수 작성
4. 문자열을 뒤집는 함수 작성
