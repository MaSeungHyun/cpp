# 05. 클래스와 객체

## 학습 목표

- 객체 지향 프로그래밍의 기초 이해
- 클래스와 객체의 개념 학습
- 생성자와 소멸자 이해
- 접근 제어와 캡슐화 학습

## 학습 내용

### 1. 클래스 기초

- 클래스 정의
- 멤버 변수와 멤버 함수
- 객체 생성과 사용
- this 포인터

### 2. 생성자와 소멸자

- 생성자 (Constructor)
- 기본 생성자
- 매개변수가 있는 생성자
- 소멸자 (Destructor)

### 3. 접근 제어

- private, public, protected
- getter와 setter
- 캡슐화 (Encapsulation)

### 4. 고급 기능

- static 멤버
- const 멤버 함수
- friend 함수
- 복사 생성자

## 예제 파일

- `01_basic_class.cpp` - 기본 클래스
- `02_constructors.cpp` - 생성자와 소멸자
- `03_encapsulation.cpp` - 캡슐화
- `04_static_members.cpp` - static 멤버
- `05_copy_constructor.cpp` - 복사 생성자

## 연습 문제

1. 은행 계좌 클래스 만들기
2. 학생 정보 관리 클래스
3. 간단한 도서 관리 시스템
4. 날짜 클래스 구현
