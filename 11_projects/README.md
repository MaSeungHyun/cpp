# 11. 종합 프로젝트

## 학습 목표

- 배운 내용을 종합하여 실전 프로젝트 구현
- 객체 지향 설계 연습
- 파일 입출력과 데이터 관리

## 프로젝트 아이디어

### 1. 학생 관리 시스템

- 학생 정보 추가, 삭제, 수정, 검색
- 성적 관리 및 통계
- 파일에 저장 및 로드
- 성적순 정렬

### 2. 도서 관리 시스템

- 도서 등록 및 검색
- 대출/반납 시스템
- 회원 관리
- 파일 기반 데이터베이스

### 3. 간단한 게임

- 텍스트 기반 RPG
- 퍼즐 게임
- 틱택토
- 행맨 게임

### 4. 유틸리티 프로그램

- 일정 관리 프로그램
- 가계부
- 주소록
- 메모장

## 프로젝트 구조

```
project/
├── main.cpp          # 메인 프로그램
├── classes/
│   ├── Student.h     # 클래스 헤더
│   └── Student.cpp   # 클래스 구현
├── utils/
│   ├── FileHandler.h # 유틸리티
│   └── FileHandler.cpp
└── data/
    └── students.dat  # 데이터 파일
```

## 개발 단계

1. **요구사항 분석**: 무엇을 만들지 정의
2. **설계**: 클래스 구조 설계
3. **구현**: 단계별로 기능 구현
4. **테스트**: 각 기능 테스트
5. **개선**: 버그 수정 및 기능 추가

## 권장사항

- 작은 기능부터 시작
- 코드를 모듈화
- 주석 작성
- 예외 처리
- 사용자 친화적인 UI

## 예제 프로젝트

- `student_management.cpp` - 학생 관리 시스템
- `simple_game.cpp` - 간단한 게임
