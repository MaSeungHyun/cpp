# 01. C++ 기초

## 학습 목표

- C++의 기본 문법 이해
- 변수와 자료형 이해
- 입출력 방법 학습
- 기본 연산자 사용

## 학습 내용

### 1. Hello World

- 첫 C++ 프로그램 작성
- iostream 헤더 이해
- main 함수의 역할

### 2. 변수와 자료형

- 정수형: int, long, short
- 실수형: float, double
- 문자형: char
- 불린형: bool
- 문자열: string

### 3. 입출력

- cout을 이용한 출력
- cin을 이용한 입력
- printf/scanf (C 스타일)

### 4. 연산자

- 산술 연산자: +, -, \*, /, %
- 비교 연산자: ==, !=, <, >, <=, >=
- 논리 연산자: &&, ||, !
- 대입 연산자: =, +=, -=, \*=, /=

## 예제 파일

- `01_hello_world.cpp` - Hello World 프로그램
- `02_variables.cpp` - 변수와 자료형
- `03_input_output.cpp` - 입출력 예제
- `04_operators.cpp` - 연산자 사용

## 연습 문제

1. 두 수를 입력받아 사칙연산 결과를 출력하는 프로그램 작성
2. 이름과 나이를 입력받아 "안녕하세요, [이름]님! [나이]살이시군요!" 출력
3. 섭씨 온도를 입력받아 화씨로 변환하는 프로그램 작성
