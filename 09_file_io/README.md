# 09. 파일 입출력

## 학습 목표

- 파일 읽기와 쓰기
- 텍스트 파일 처리
- 바이너리 파일 처리
- 파일 스트림 활용

## 학습 내용

### 1. 텍스트 파일

- ifstream (읽기)
- ofstream (쓰기)
- fstream (읽기/쓰기)

### 2. 파일 모드

- ios::in, ios::out
- ios::app (추가)
- ios::binary

### 3. 파일 처리

- 줄 단위 읽기
- 단어 단위 읽기
- 파일 존재 확인
- 파일 위치 제어

## 예제 파일

- `01_text_file.cpp` - 텍스트 파일
- `02_csv_file.cpp` - CSV 파일
- `03_binary_file.cpp` - 바이너리 파일

## 연습 문제

1. 학생 정보를 파일에 저장하고 불러오기
2. 로그 파일 작성 프로그램
3. 간단한 데이터베이스 시스템
