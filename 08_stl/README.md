# 08. STL (Standard Template Library)

## 학습 목표

- STL 컨테이너 사용법 학습
- 반복자(Iterator) 이해
- STL 알고리즘 활용
- 함수 객체와 람다 함수

## 학습 내용

### 1. 시퀀스 컨테이너

- vector
- list
- deque

### 2. 연관 컨테이너

- map
- set
- unordered_map
- unordered_set

### 3. STL 알고리즘

- sort, find, count
- accumulate
- for_each
- transform

### 4. 기타

- pair, tuple
- iterator
- 람다 함수

## 예제 파일

- `01_vector.cpp` - vector 사용법
- `02_map_set.cpp` - map과 set
- `03_algorithms.cpp` - STL 알고리즘
- `04_lambda.cpp` - 람다 함수

## 연습 문제

1. vector를 이용한 학생 관리 프로그램
2. map을 이용한 단어 빈도 계산
3. STL 알고리즘을 이용한 데이터 처리
