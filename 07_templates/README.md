# 07. 템플릿

## 학습 목표

- 템플릿의 개념과 필요성 이해
- 함수 템플릿 작성
- 클래스 템플릿 작성
- 템플릿 특수화

## 학습 내용

### 1. 함수 템플릿

- 일반 함수 템플릿
- 여러 타입 매개변수
- 템플릿 오버로딩

### 2. 클래스 템플릿

- 기본 클래스 템플릿
- 템플릿 멤버 함수
- 컨테이너 클래스

### 3. 고급 주제

- 템플릿 특수화
- 부분 특수화
- 가변 인자 템플릿

## 예제 파일

- `01_function_templates.cpp` - 함수 템플릿
- `02_class_templates.cpp` - 클래스 템플릿
- `03_template_specialization.cpp` - 템플릿 특수화

## 연습 문제

1. 제네릭 배열 클래스 만들기
2. 스택 자료구조 템플릿 구현
3. 다양한 타입 지원하는 유틸리티 함수
