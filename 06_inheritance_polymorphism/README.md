# 06. 상속과 다형성

## 학습 목표

- 상속의 개념과 활용
- 가상 함수와 다형성 이해
- 추상 클래스와 인터페이스
- 연산자 오버로딩

## 학습 내용

### 1. 상속 기초

- 상속의 개념
- protected 접근 제어자
- 생성자와 소멸자 호출 순서
- 다중 상속

### 2. 다형성

- 가상 함수 (virtual)
- 순수 가상 함수
- 추상 클래스
- 동적 바인딩

### 3. 연산자 오버로딩

- 산술 연산자
- 비교 연산자
- 입출력 연산자
- 증감 연산자

## 예제 파일

- `01_basic_inheritance.cpp` - 상속 기초
- `02_virtual_functions.cpp` - 가상 함수
- `03_abstract_class.cpp` - 추상 클래스
- `04_operator_overloading.cpp` - 연산자 오버로딩

## 연습 문제

1. 도형 클래스 계층 구조 만들기
2. 동물 클래스와 상속
3. 복소수 클래스와 연산자 오버로딩
