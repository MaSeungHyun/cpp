# 02. 제어문

## 학습 목표

- 조건문을 사용하여 프로그램의 흐름 제어
- 반복문을 사용하여 코드 재사용
- break와 continue 이해

## 학습 내용

### 1. 조건문 (if-else)

- if 문
- if-else 문
- if-else if-else 문
- 중첩 if 문

### 2. switch 문

- switch-case 구조
- break의 역할
- default 케이스

### 3. 반복문

- for 문
- while 문
- do-while 문
- 중첩 반복문

### 4. 제어 키워드

- break: 반복문 탈출
- continue: 다음 반복으로
- goto (권장하지 않음)

## 예제 파일

- `01_if_else.cpp` - 조건문 예제
- `02_switch.cpp` - switch 문 예제
- `03_for_loop.cpp` - for 반복문
- `04_while_loop.cpp` - while 반복문
- `05_nested_loop.cpp` - 중첩 반복문

## 연습 문제

1. 점수를 입력받아 학점(A, B, C, D, F)을 출력하는 프로그램
2. 1부터 100까지의 합을 구하는 프로그램
3. 구구단을 출력하는 프로그램
4. 소수(prime number)를 판별하는 프로그램
